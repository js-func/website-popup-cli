# website-popup-cli

> Open a website in a popup *(macOS)*

<img src="screenshot.png" width="759">

## Install

```
$ npm install --global website-popup-cli
```

## Usage

```
$ website-popup --help

  Usage
    $ website-popup <url> [--size=<size>]

  Example
    $ website-popup https://sindresorhus.com --size=600x400
```
